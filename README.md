# data-visualization-project
# Student Social Media Addiction Analysis

## Overview
This project analyzes social media usage patterns among students using R. The goal was to investigate whether high usage correlates with addiction levels, sleep reduction, and emotional content exposure.

## Research Questions
- How many hours per day do students spend on social media?
- What platforms dominate student usage?
- Do students show signs of addictive behavior?
- Is there a correlation between addiction score and sleep?
- What emotional tone dominates social media content?

## Tools Used
- R
- tidyverse
- ggplot2
- wordcloud
- bing sentiment lexicon

## Key Findings
- Students average nearly 5 hours per day online
- Instagram and TikTok dominate usage
- Addiction scores skew high (7–9 range)
- Negative correlation between addiction score and sleep
- Emotional and negative content increases engagement

## Project Structure
- `/data` → Raw datasets
- `/code` → R scripts used for analysis
- `/output` → Visualizations
